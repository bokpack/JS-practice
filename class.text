#객체지향프로그래밍언어 들은 #클래스 라는 문법으로 #객체 를 효율적이고 안전하게 만듦.
#추상화 란 어떤 객체의 공통사항을 추출하는 것을



\ 이스케이프 문자(escape character)
\t 탭
\n 줄바꿈



`` 템플릿 리터럴 (템플릿 문자열) : 문자열 안에 변수나 표현식을 쉽게 삽입 가능
${} 내부에는 변수나 연산이 들어갈 수 있음 변수나 표현식을 문자열에 삽입

#메소드(method) 란 객체내에서 정의된 함수
#생성자(construct) 란 클래스를 기반으로 인스턴스를 생성할 떄 처음 호출되는 메소드
생성자에게는 속성을 추가하는 등 객체의 초기화 처리를 함
